Neste desafio, construi um website de um restaurante fictício chamado "Massa!", onde nele é possível obter informações gerais do restaurante e montar o seu pedido em uma seção separada.

Este website foi desenvolvido em HTML, CSS, Boostrap 5 e Javascript/JQuery.

obs. Como o Gmail não permite enviar arquivos ".js", não pude deixar o código JS em um arquivo separado. Por isso, ele se encontra dentro da tag "script" do arquivo HTML.
